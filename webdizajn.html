Osnove HTML-a i CSS-a

Učimo osnovne koncepte web dizajna koristeći HTML i CSS. Ovo uključuje strukturu web stranice, stilizaciju elemenata, pozicioniranje, boje, fontove i mnogo više.

Struktura HTML-a

HTML (HyperText Markup Language) je osnovni jezik za izgradnju web stranica. Sastoji se od elemenata koji opisuju sadržaj stranice.

Stilizacija CSS-om

CSS (Cascading Style Sheets) se koristi za stilizaciju HTML elemenata. Ovo uključuje boje, fontove, pozadine, marginu, padding, i mnogo više.

Web Dizajn u Praksi

U praktičnom dijelu učimo kako koristiti HTML i CSS za izradu modernih i funkcionalnih web stranica. Ovo uključuje izradu layouta, navigacije, forma, i druge elemente.
